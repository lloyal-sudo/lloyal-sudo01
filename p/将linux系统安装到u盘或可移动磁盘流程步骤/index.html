将Linux系统安装到U盘或可移动磁盘流程步骤

2026-03-23

按照 准备工具 🛠️ -> 制作启动介质 🗂️ -> 安装系统 💻 -> post-install配置 🚀 四个阶段来执行。

为了确保成功率，我以最通用的 Windows 主机 + Ventoy（多系统管理神器） 为例。

第一阶段：极致准备工作 (0% 风险)

这一步最关键，决定了后面会不会“变砖”。

  1. 磁盘容量确认

    • U盘 (安装介质)：至少 8GB，建议 16GB+。
    • 移动硬盘/U盘 (系统目标)：
      • 最低要求：20GB（仅命令行系统）。
      • 推荐要求：64GB（如果想装图形界面、开发环境和各种工具）。
    • ⚠️ 警告：目标盘所有数据将被彻底清除，请先备份。

  2. 下载必备工具

    1. CentOS 镜像：确认你的 CentOS-Stream-10-latest-x86_64-dvd1.iso 下载完整。
    2. Ventoy2Disk（下载地址：ventoy.net）：这是比 Rufus 更好的工具，制作好后，U盘还能存其他文件，且支持直接放多个 ISO 启动。
    3. 校验工具（可选）：Windows 自带 PowerShell，用于验证镜像是否损坏。

  3. 验证镜像完整性 (必做)

    • 打开 PowerShell，进入你放 ISO 文件的目录，执行命令： 
      1

      Get-FileHash -Path .\CentOS-Stream-10-latest-x86_64-dvd1.iso -Algorithm SHA256
    • 对比生成的哈希值与官网提供的 SHA256 值。不一致会导致安装失败，必须重新下载。

  4. 整理目标盘

    • 将准备好的 大容量移动磁盘 (U盘/SSD) 插入电脑。
    • 打开 “此电脑”，记住它的盘符（例如 E: 或 F:），后续安装时千万别选错！

第二阶段：制作启动 U 盘 (10% 风险)

使用 Ventoy，傻瓜式操作，容错率极高。

  1. 运行 Ventoy

    • 解压下载的 Ventoy，运行其中的 Ventoy2Disk.exe。
    • 设备 (Device)：选择你的小容量 U 盘（注意：不要选错成目标移动硬盘！）。

  2. 配置参数

    • 分区类型：
      • 如果你的电脑是近5年买的，选 GPT。
      • 如果是很老的电脑，选 MBR。
    • 其他默认，不要动 Secure Boot 相关。

  3. 写入启动项

    • 点击 「安装」。
    • 等待进度条走完，U盘会自动弹出。这时候 U 盘里只有几个 Ventoy 的引导文件，空间还很大。

  4. 放入镜像

    • 重新插入 U 盘。
    • 把下载好的 CentOS-Stream-10...iso 文件直接拖拽到 U 盘的根目录下。
    • ✅ 完成！ 你的 U 盘现在就是一个万能系统启动盘了。

第三阶段：进入安装界面 (30% 风险)

这里最容易出错的是“从U盘启动”。

  1. 插入介质

    • 把 Ventoy U 盘 插在电脑上。
    • 把 目标移动硬盘 也插在电脑上（最好插在主机背面的 USB 3.0 接口，速度快）。

  2. 设置启动顺序

    • 重启电脑。
    • 开机瞬间疯狂按 启动项快捷键（台式机通常是 F12、F11 或 Esc；笔记本通常是 F2、F10）。
    • 在启动菜单里，选择 UEFI USB Key 相关的选项。

  3. 选择启动项

    • 进入 Ventoy 界面后，会列出 U 盘里的 ISO 文件。
    • 用方向键选中 CentOS-Stream-10...iso，按回车。

第四阶段：系统安装详解 (50% 风险)

这是最细化的步骤，跟着做就行。

  1. 语言与键盘

    • 推荐选 English - English (United States)（环境变量更标准，少坑）。
    • 点击 Continue（继续）。

  2. 进入安装摘要界面

    • 屏幕中间会有一个大的 “INSTALLATION SUMMARY”。
    • 这里有 3 个地方必须配置，其余默认即可。

  3. 关键配置 1：安装目标 (重中之重！)

    • 点击 **Installation Destination (安装目的地)。
    • 选择磁盘：在列表中找到你的 移动硬盘（通常显示为 128GB/500GB 的外部磁盘）。
    • 勾选它。
    • Other Storage Options (其他存储选项) -> 选择 Custom (Manual partitioning) (手动分区)。
    • 点击 Done (完成)。

  4. 关键配置 2：手动分区 (核心难点)

    • 进入分区界面，点击 + 号新建挂载点。
    • 我们需要建 3 个分区：
      • 1. 引导分区 (EFI)
        • Mount Point: 选择 /boot/efi
        • Desired Capacity: 输入 600 (单位MB)
        • 点击 Add Mount Point
      • 2. 交换分区 (Swap)
        • Mount Point: 选择 swap
        • Desired Capacity: 输入 16384 (单位MB，即16GB，物理内存8G以上建议固定给16G)
        • 点击 Add Mount Point
      • 3. 根分区 (Root)
        • Mount Point: 输入 /
        • Desired Capacity: 保持为空，直接点击 Add Mount Point (它会自动吃掉剩下所有空间)。
    • ✅ 检查左侧列表：有 /boot/efi, swap, / 三个条目。
    • 点击 Done。
    • 弹出摘要框，点击 Accept Changes (接受更改)。

  5. 关键配置 3：网络与主机名

    • 回到主菜单，点击 Network & Host Name (网络和主机名)。
    • 选中 Ethernet (以太网)，点击右上角的 ON (打开)。
    • 确保能获取到 IP 地址（显示 192.168.x.x），这样安装完就能上网更新软件。
    • 点击 Done。

  6. 开始安装

    • 所有配置变红的地方都消失后，点击右上角的 Begin Installation (开始安装)。

  7. 设置用户密码

    • 安装过程中，会有 Root Password (根密码) 和 Create User (创建用户) 两个环节。
    • Root Password：设置强密码（记得住的，比如 Admin@123456）。
    • Create User：创建一个日常使用的普通用户。
    • 耐心等待进度条走完。

  8. 重启系统

    • 点击 Reboot System (重启系统)。
    • 重启时，拔掉 Ventoy U 盘。
    • 系统会从你的 移动硬盘 启动 CentOS Stream 10。

第五阶段：Post-install 配置 (100% 完成)

首次启动可能需要 1-2 分钟，请耐心等待。

  1. 登录系统

    • 输入你刚才创建的用户名和密码登录。

  2. 联网验证

    • 尝试上网： 
      1

      ping www.baidu.com
    • 如果通了，说明环境完美。

  3. 更新系统

    • 这是最后一步，确保系统最新： 
      1

      sudo dnf update -y
    • 等待命令执行完毕（可能需要几分钟）。

🎉 恭喜！你现在拥有了一个完美的 CentOS Stream 10 移动系统！

“单盘启动模式”。

一、什么是“单盘启动”？

简单说，就是你把 CentOS-Stream-10-latest-x86_64-dvd1.iso 直接“写入”到你的可移动磁盘里，让它变成一个可直接启动的系统盘。

核心区别：

  • 之前的方案：U盘做引导盘 + 移动磁盘存系统（两个盘）。
  • 现在的方案：只插一个盘，它既是引导盘，也是系统盘（一个盘搞定）。

二、操作核心步骤（Windows 环境）

我们用 Ventoy 或 Rufus 工具，直接把 ISO 写入磁盘。

1. 准备工作

  • 可移动磁盘：建议 ≥32GB（U盘或移动SSD均可）。
  • 数据备份：磁盘内所有数据将被彻底清除，请先备份！
  • 工具：下载 Ventoy 或 Rufus。

2. 制作单盘启动系统（以 Ventoy 为例）

  1. 打开 Ventoy2Disk.exe。
  2. 设备 (Device)：选择你的可移动磁盘（注意：不要选错成电脑硬盘！）。
  3. 分区类型：
    • 电脑较新（近5年）：选 GPT。
    • 电脑很老：选 MBR。
  4. 点击 「安装」，等待完成。
  5. 把下载好的 CentOS-Stream-10-latest-x86_64-dvd1.iso 直接拖拽到这个磁盘里。
  6. ✅ 完成！ 现在你只有这一个盘，但它已经是一个能启动 Linux 的系统盘了。

三、如何使用（单盘启动）

  1. 只插这一个可移动磁盘 到任意电脑。
  2. 重启电脑，开机狂按 启动项快捷键（F12/F11/Esc 等）。
  3. 在启动菜单里，选择 UEFI USB Key 或 USB Hard Disk 相关选项（取决于磁盘格式）。
  4. 电脑会直接从这个U盘里的 CentOS 系统启动。
  5. 进入安装界面后，安装目的地 依然选择这个可移动磁盘，并完成分区（参考之前给你的 /boot/efi + swap + / 方案）。
  6. 安装完成，重启。依然只插这个盘，即可进入你的 CentOS 系统。

四、关键注意事项

  1. 磁盘容量：
    • 用 U 盘做系统盘：建议 32GB 以上，体验更好（安装后系统+软件占用约20GB+）。
    • 用 移动SSD 做系统盘：体验最佳，速度快，寿命长。
  2. 启动模式：
    • 制作时的分区类型（GPT/MBR）必须与电脑的启动模式（UEFI/Legacy）匹配，否则无法启动。
  3. 性能提醒：
    • 用普通 U 盘做系统盘，日常办公、学习编程、搭建靶场完全够用。但如果进行大量文件读写或复杂编译，速度会比电脑内置硬盘慢一些。
