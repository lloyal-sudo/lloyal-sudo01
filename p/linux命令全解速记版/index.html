Linux命令全解速记版

2026-03-20

一、文件与目录

  • ls 列出文件
    • -l 详细 -a 隐藏 -h 易读 -t 按时间 -r 倒序
  • cd 切换目录
    • cd .. 上一级 cd ~ 家目录 cd - 回到上一次目录
  • pwd 显示当前路径
  • mkdir 创建目录
    • mkdir -p 递归多级
  • rmdir 删除空目录
  • touch 创建空文件/改时间
  • cp 复制
    • cp -r 递归目录 -i 提示覆盖 -p 保留权限
  • mv 移动/重命名
  • rm 删除（慎用）
    • -r 目录 -f 强制 -i 询问

二、文件查看与编辑

  • cat 查看全文
  • tac 倒着看
  • more 分页往下
  • less 上下翻页、搜索（/关键词）
  • head 看开头
    • head -n 20 前20行
  • tail 看结尾
    • tail -n 20 后20行
    • tail -f 实时刷新日志
  • nl 带行号显示
  • od 二进制查看
  • vim/vi 编辑器
    • i 编辑 esc 退出编辑
    • :w 保存 :q 退出 :wq 保存退出 :q! 强制不保存

三、文件查找与搜索

  • which 找命令路径
  • whereis 找命令、源码、帮助
  • locate 快速查找文件（需数据库）
  • find 精准查找
    • find / -name 文件名
    • find / -type f 只找文件
    • find / -size +100M 大于100M
    • find / -user root 属主root
    • find / -mtime -7 7天内修改
  • grep 过滤内容
    • -i 忽略大小写 -v 反选 -n 显示行号 -c 统计行数 -r 递归搜目录
  • awk 列截取、统计
  • sed 替换、删除行

四、权限、用户、组

权限

  • chmod 修改权限
    • 数字：777、755、644
    • 符号：u+x、g-w、a=r
  • chown 改属主/属组
    • chown user:group file
  • chgrp 改属组
  • umask 默认权限掩码

用户/组

  • whoami 我是谁
  • who 谁在线
  • w 详细在线用户
  • useradd 新建用户
  • userdel -r 删除用户并删家目录
  • usermod 修改用户
  • groupadd 建组
  • groupdel 删组
  • passwd 改密码
  • id 查看uid、gid
  • su 切换用户
  • sudo 临时root权限

五、磁盘与挂载

  • df -h 磁盘使用
  • du -sh 目录大小
  • du -sh * 看每个文件/目录大小
  • lsblk 磁盘列表
  • blkid 查看UUID
  • mount 挂载
  • umount 卸载
  • fdisk 磁盘分区（MBR）
  • parted 分区（GPT）
  • mkfs.ext4 格式化ext4
  • mkfs.xfs 格式化xfs
  • fsck 磁盘检查修复

六、进程与系统资源

  • ps
    • ps aux 所有进程
    • ps -ef 完整格式
  • top 实时进程
  • htop 增强版top
  • uptime 运行时间、负载
  • free -h 内存
  • kill PID 结束进程
  • kill -9 PID 强制杀
  • killall 按名字杀
  • pkill 按名字杀
  • bg 后台挂起
  • fg 切前台
  • jobs 后台任务
  • nohup 后台挂机运行

七、压缩解压

  • tar
    • 压缩：tar -zcvf xxx.tar.gz 目录
    • 解压：tar -zxvf xxx.tar.gz
    • -C 指定解压目录
  • zip 压缩zip
  • unzip 解压zip
  • gzip
  • gunzip
  • bzip2
  • bunzip2
  • xz
  • unxz

八、网络命令（你之前缺的很多在这）

  • ip addr 看IP
  • ifconfig 旧版IP
  • ip route 路由
  • ping 通断测试
  • telnet IP 端口 测端口
  • ssh 用户名@IP 远程登录
  • scp 跨机传文件
  • sftp 安全传输
  • netstat -tulnp 看端口
  • ss -tulnp 更快查端口
  • ss -s 网络统计
  • nslookup 解析域名
  • dig 详细DNS
  • host 域名解析
  • curl 请求网页/接口
  • wget 下载文件
  • ethtool 网卡信息
  • traceroute 追踪路由
  • mtr 综合ping+路由
  • firewall-cmd 防火墙（CentOS7+）
  • ufw 防火墙（Ubuntu）
  • iptables 防火墙规则

九、系统、服务、日志

  • uname -a 内核信息
  • cat /etc/os-release 系统版本
  • date 时间
  • timedatectl 时间时区
  • hostname 主机名
  • reboot 重启
  • halt 关机
  • shutdown -h now 立刻关机
  • history 历史命令
  • clear 清屏
  • alias 别名
  • unalias 取消别名

服务管理

  • systemctl start 服务
  • systemctl stop 服务
  • systemctl restart 服务
  • systemctl status 服务
  • systemctl enable 服务 开机自启
  • systemctl disable 服务 关闭自启

日志

  • dmesg 内核日志
  • journalctl 系统日志
    • -f 实时 -u 看某个服务

十、IO、重定向、管道

  • > 覆盖重定向
  • >> 追加
  • < 输入重定向
  • | 管道
  • xargs 传参
  • tee 同时输出到文件+屏幕

十一、磁盘、文件高级

  • ln 链接

  • diff 对比文件
  • md5sum 校验md5
  • sha256sum 校验sha256
  • dd 拷贝镜像、磁盘

十二、性能、监控

  • iostat IO性能
  • iotop IO进程top
  • vmstat 虚拟内存、CPU
  • sar 系统统计
  • mpstat CPU统计
