使用Git+hugo+GitHub 0成本搭建个人博客平台

2026-03-14

整体思路：

环境准备（Git+Hugo）→ 本地创建博客 → 代码托管（GitHub）→ 配置 GitHub Pages 实现在线访问。全程无需任何费用，只需基础的命令行操作和 GitHub 账号。

1、环境准备

（1）、安装 Git

Windows：官网下载 Git for Windows，安装时默认选项即可，完成后打开 Git Bash 验证：

git –version

输出git版本号即安装成功

（2）、安装 Hugo（推荐安装_extended 版本，支持更多主题）

下载 Hugo Extended（选择对应 Windows 版本，如hugo_extended_0.120.0_windows-amd64.zip），解压后将hugo.exe所在文件夹路径添加到系统环境变量（如D:\hugo\bin）。 验证：

hugo version # 输出hugo版本，且包含"extended"即成功

（3）、GitHub 账号准备

注册 GitHub 账号（官网）

2. 本地创建 Hugo 博客

（1）初始化博客项目

打开 Git Bash，选择一个本地文件夹，执行：

进入目标文件夹

cd （目标文件）

创建hugo博客项目（blog为项目名，可自定义）

hugo new site blog

进入项目目录

cd blog

执行后会生成 Hugo 标准目录结构，核心目录说明：

content/：存放博客文章（Markdown 格式）；

themes/：存放博客主题；

config.toml：博客核心配置文件（站点名称、主题、语言等）。

（2）安装 Hugo 主题（以主流的 LoveIt 为例）

克隆主题到themes目录

git clone https://github.com/dillonzq/LoveIt.git themes/LoveIt

复制主题示例配置（方便快速上手）

cp themes/LoveIt/exampleSite/config.toml ./

*** Hugo 有大量免费主题，可在 Hugo Themes 挑选 ***

（3）本地创建第一篇文章并预览

创建第一篇文章（content/posts/ 是文章存放路径，first-post.md是文件名）

hugo new posts/first-post.md

启动本地预览服务器(-D 表示显示草稿文章)

hugo server -D

3. 关联 GitHub 仓库（代码托管）

（1）创建 GitHub 仓库

仓库名必须为：你的GitHub用户名.github.io（例如：xiaoming.github.io），否则后续 Pages 无法访问；Public（必须公开）

（2）本地博客项目关联 GitHub 仓库

回到本地 Git Bash / 终端，在博客目录（blog/）执行：

初始化git仓库（若未初始化）

git init

添加远程仓库（替换为你的仓库地址）

git remote add origin https://github.com/你的用户名/你的 用户名.github.io.git

生成博客静态文件（关键：Hugo将md转为静态html）执行后会生成public/目录，存放所有静态文件

hugo -D

进入public目录（该目录是最终要部署的内容）

cd public

提交public目录到GitHub

#public目录单独初始化git（避免嵌套仓库）

git init

git add .

git commit -m “first commit: hugo blog static files”

推送到GitHub（main分支，GitHub默认分支为main）

git branch -M main

git push -u origin main

4. 配置 GitHub Pages（实现在线访问）

进入 GitHub 仓库页面 → 点击「Settings」→ 左侧栏「Pages」；

「Build and deployment」→「Source」选择「Deploy from a branch」；

「Branch」选择「main」，「Folder」选择「/(root)」，点击「Save」；

等待 1-2 分钟，页面会显示「Your site is live at https:// 你的用户名.github.io/」，访问该地址即可看到在线博客。

5. 后续维护（发布新文章）

每次写新文章后，重复以下步骤即可更新在线博客：

本地创建新文章

hugo new posts/新文章.md

编辑文章后，重新生成静态文件

hugo -D

进入public目录，提交并推送

cd public

git add .

git commit -m “add new post: 新文章标题”

git push origin main

至此你成功的搭建了一个个人博客。
