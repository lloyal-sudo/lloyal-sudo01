一、前置准备：搭建 PHP 运行环境

1. 推荐集成环境（二选一）

  • WAMP（仅 Windows）：官网下载

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>文章列表</title>
</head>
<body>
    <h1>文章列表</h1>
    <?php
        // 1. 配置数据库连接信息
        $host = 'localhost'; // 数据库地址
        $user = 'root'; // 数据库用户名（默认root）
        $password = ''; // 数据库密码（默认空）
        $dbname = 'php_website'; // 数据库名

        // 2. 连接数据库（使用 mysqli 扩展，推荐）
        $conn = new mysqli($host, $user, $password, $dbname);

        // 3. 检查连接是否成功
        if ($conn->connect_error) {
            die("数据库连接失败: " . $conn->connect_error);
        }

        // 4. 设置字符编码（避免中文乱码）
        $conn->set_charset("utf8mb4");

        // 5. 执行查询语句
        $sql = "SELECT id, title, content, create_time FROM articles";
        $result = $conn->query($sql);

        // 6. 处理查询结果
        if ($result->num_rows > 0) {
            // 循环输出每篇文章
            while($row = $result->fetch_assoc()) {
                echo "<div style='margin: 20px 0; padding: 10px; border: 1px solid #ccc;'>";
                echo "<h2>{$row['title']}</h2>";
                echo "<p>{$row['content']}</p>";
                echo "<p>发布时间：{$row['create_time']}</p>";
                echo "</div>";
            }
        } else {
            echo "<p>暂无文章</p>";
        }

        // 7. 关闭数据库连接
        $conn->close();
    ?>
</body>
</html>

运行效果

访问 http://localhost/article_list.php，能看到从 MySQL 中查询出的文章列表，实现了动态数据展示。

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>发布文章</title>
</head>
<body>
    <h1>发布新文章</h1>
    <!-- 表单 -->
    <form method="POST" action="">
        <div style="margin: 10px 0;">
            <label>标题：</label>
            <input type="text" name="title" required style="width: 300px; padding: 5px;">
        </div>
        <div style="margin: 10px 0;">
            <label>内容：</label>
            <textarea name="content" required rows="10" cols="50"></textarea>
        </div>
        <button type="submit" name="submit">提交</button>
    </form>

    <?php
        // 处理表单提交
        if (isset($_POST['submit'])) {
            // 获取表单数据
            $title = $_POST['title'];
            $content = $_POST['content'];

            // 数据库连接（同上文）
            $host = 'localhost';
            $user = 'root';
            $password = '';
            $dbname = 'php_website';
            $conn = new mysqli($host, $user, $password, $dbname);

            if ($conn->connect_error) {
                die("连接失败: " . $conn->connect_error);
            }
            $conn->set_charset("utf8mb4");

            // 预处理语句（防止SQL注入）
            $stmt = $conn->prepare("INSERT INTO articles (title, content) VALUES (?, ?)");
            $stmt->bind_param("ss", $title, $content); // ss 表示两个字符串参数

            // 执行插入
            if ($stmt->execute()) {
                echo "<p style='color: green;'>文章发布成功！<a href='article_list.php'>查看文章列表</a></p>";
            } else {
                echo "<p style='color: red;'>发布失败：" . $stmt->error . "</p>";
            }

            // 关闭连接
            $stmt->close();
            $conn->close();
        }
    ?>
</body>
</html>

总结

  1. 环境搭建：新手优先用 XAMPP/WAMP 集成环境，一键启动 Apache + MySQL，无需手动配置；
  2. 核心逻辑：PHP 嵌入 HTML 中运行，通过 echo 输出动态内容，通过 mysqli 扩展连接 MySQL 实现数据交互；
  3. 安全要点：数据库操作使用预处理语句（prepare/bind_param）防止 SQL 注入，处理用户输入时做合法性校验。

补充：XAMPP介绍

一、什么是集成环境（Web 开发视角）

集成环境（这里特指 PHP 开发的集成环境）是将搭建 PHP 网站所需的所有核心组件打包在一起的软件包，无需你手动分别安装、配置 Web 服务器、PHP 解释器、数据库等，一键即可完成环境部署。

可以用一个比喻理解：

  • 手动搭建环境 = 自己买CPU、主板、内存等配件，一步步组装电脑；
  • 集成环境 = 直接买一台预装系统和常用软件的整机，开箱即用。

二、XAMPP 详解

XAMPP 是目前最流行的跨平台 PHP 集成环境，名字是各核心组件的缩写：

  • X：Cross-platform（跨平台），支持 Windows、Mac、Linux；
  • A：Apache（Web 服务器），负责接收浏览器请求、返回网页内容；
  • M：MySQL/MariaDB（数据库），存储网站数据（如用户信息、文章内容）；
  • P：PHP（脚本解释器），处理动态逻辑、与数据库交互；
  • P：Perl（可选脚本语言，新手可忽略）。

1. XAMPP 的核心特点

  • 零配置：安装后无需修改复杂的配置文件（如 Apache 的 httpd.conf、PHP 的 php.ini），默认配置即可满足新手开发需求；
  • 一键启停：通过控制面板可快速启动/停止 Apache、MySQL 等服务；
  • 自带工具：内置 phpMyAdmin（可视化管理 MySQL 数据库）、FileZilla（FTP 服务器）等实用工具；
  • 完全免费：开源软件，无付费门槛。

2. XAMPP 的关键使用细节

（1）安装与启动
  • 下载：从 Apache Friends 官网 下载对应系统的版本；
  • 安装：一路点击「Next」即可（建议安装路径不要包含中文/空格，如 D:\xampp）；
  • 启动：打开 XAMPP Control Panel，点击 Apache 和 MySQL 后的「Start」，出现绿色「Running」即表示启动成功。
（2）核心目录说明
目录路径 作用
xampp/htdocs 网站根目录（所有 PHP/HTML 文件放这里）
xampp/mysql MySQL 数据库相关文件
xampp/php PHP 解释器及配置文件（php.ini）
xampp/apache Apache 服务器及配置文件
（3）验证 XAMPP 是否可用

启动 Apache 后，打开浏览器访问 http://localhost 或 http://127.0.0.1，能看到 XAMPP 的欢迎页面，说明环境正常； 访问 http://localhost/phpmyadmin 可进入数据库管理界面，验证 MySQL 是否可用。

（4）常见问题解决
  • Apache 启动失败：大概率是 80 端口被占用（如 Windows 的 IIS、微信/QQ 等软件），可在 XAMPP 控制面板中点击 Apache 后的「Config」→「Apache (httpd.conf)」，修改 Listen 80 为 Listen 8080，重启后访问 http://localhost:8080；
  • MySQL 启动失败：通常是 3306 端口被占用，或之前安装过 MySQL 残留服务，可在控制面板中修改 MySQL 端口，或卸载残留服务。

总结

  1. 集成环境：打包了 PHP 网站所需的 Web 服务器、解释器、数据库等组件，新手无需手动配置，开箱即用；
  2. XAMPP：跨平台的 PHP 集成环境，核心包含 Apache+MySQL+PHP，根目录是 htdocs，启动后通过 localhost 访问，是新手入门的首选。
